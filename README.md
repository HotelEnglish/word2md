# Word to Markdown Converter

一个简单高效的在线Word文档转Markdown工具。支持直接粘贴Word内容或上传Word文件，自动转换为Markdown格式。

## 功能特点

- 支持直接粘贴Word内容
- 支持上传Word文档（.docx格式）
- 支持表格转换
- 保留文本格式（标题、列表、粗体等）
- 实时预览转换结果
- 一键复制转换后的Markdown内容
